⏺ The uncommitted change in update_manager.rb adds "Runtime" => "runc" to the updater container's HostConfig. Without it, the updater container
  would inherit the system's default runtime — which on this host is sysbox-runc. The updater just needs to run docker compose up -d, so it
  doesn't need Sysbox's nested-container capabilities, and using sysbox-runc for a simple one-shot container could cause issues or unnecessary
  overhead.
