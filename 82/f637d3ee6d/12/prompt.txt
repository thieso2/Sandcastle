## Part 3: Real Docker Integration Tests

### 3a. Lightweight Test Image

Create `images/sandbox-test/Dockerfile` — a minimal image (~100 MB):
```dockerfile
FROM ubuntu:24.04
RUN apt-get update && apt-get install -y openssh-server sudo && rm -rf /var/lib/apt/lists/*
COPY entrypoint.sh /entrypoint.sh
RUN chmod +x /entrypoint.sh
EXPOSE 22
ENTRYPOINT ["/entrypoint.sh"]
```

The entrypoint is a stripped-down version of the production one (creates user, injects SSH key, starts sshd). No Docker-in-Docker, no VNC.

### 3b. `test/integration/real_sandbox_test.rb` (new)

Skipped unless `SANDCASTLE_REAL_DOCKER=1` is set. Tests run against real Docker daemon.

**Setup:**
```ruby
skip "set SANDCASTLE_REAL_DOCKER=1 to run" unless ENV["SANDCASTLE_REAL_DOCKER"]
# Use test-specific DATA_DIR
ENV["SANDCASTLE_DATA_DIR"] = "/tmp/sandcastle-test"
# Use lightweight test image
TEST_IMAGE = ENV.fetch("SANDCASTLE_TEST_IMAGE", "sandcastle-sandbox-test:latest")
```

**Test cases:**
```
test "full sandbox lifecycle"
  → create_container_and_start (real Docker, Sysbox runtime)
  → verify container is "running" in Docker
  → verify SSH port is bound on host
  → connect_info returns valid host/port
  → stop container, verify "exited"
  → start again, verify "running"
  → create_snapshot, verify Docker image committed
  → destroy, verify container removed from Docker

test "sandbox SSH connectivity"
  → create sandbox
  → wait for sshd to start (poll port, max 30s)
  → connect via Net::SSH, run "whoami"
  → assert output == sandbox user name

test "snapshot and restore"
  → create sandbox, run echo "hello" > /test.txt in sandbox (via SSH exec)
  → create_snapshot
  → destroy original
  → create new sandbox from snapshot (image = snapshot docker_image)
  → SSH in, verify /test.txt contains "hello"

test "route creation and Traefik config"
  → create sandbox
  → create HTTP route via RouteManager
  → verify Traefik config YAML written to DATA_DIR/traefik/dynamic/
  → destroy route, verify YAML removed

test "concurrent sandbox creation"
  → create 3 sandboxes in parallel threads
  → verify all 3 get unique SSH ports
  → verify all 3 reach "running"
  → destroy all 3
```
